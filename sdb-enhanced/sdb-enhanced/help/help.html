About

SDB-Enhanced was written because the standard Neopets SDB lacks organisation and is clumsy/difficult to use. It addresses several common complaints seen on the Neoboards.

In writing this, I've been very careful not to include anything that I believe would be questionable/against the rules. The extension does not automate any player activity, or provide data that would otherwise be inaccessible; it only helps to facilitate better searching, sorting, and organisation of your SDB.

This software is not endorsed by or affliated with Neopets.

Getting Started

The first step is to visit each page of your standard SDB, so that the extension can import your item data.

To visit your standard SDB, click here or click the link to Safety Deposit Box from your inventory on Neopets.

You will see the message "Queued ## items for processing." appear at the bottom of the page. That means the items on that page are being added to your new SDB. You can click Next to go to the next page and add more items. You can stop at any page you'd like, but until you've visited each page of your standard SDB, the items will not be shown in the extension.

Once you've added as many items as you like, click the SDB icon to the right of your address bar.

Settings

You can click here to go to the settings page. You can also get to the settings page by clicking the icon.

Show Rarity

If checked, item rarity estimates (Uncommon/Rare/Very Rare) or rarities (r0, r75, r101, etc...) will be shown on the search page. See Lookup Rarities below.

Enabled

Show Descriptions

If checked, item descriptions will be shown when you move the mouse over an item.

Enabled

Maintain Search State

Keeps your search input between views of the extension. For example, if you search "Negg", then close the extension, the same search will be shown the next time you open the extension. If unchecked, the extension will show a default (empty) search form each time.

Enabled

Enable Regex Search

Adds a checkbox to the search form to make the search use a regex (regular expression). If you do not know what this is, you do not need it.

Disabled

Lookup Rarities

Uses the Neopets search box as a logged-out user to lookup item rarities.

Note: if you do not enable this, only rarity estimates will be shown. (Uncommon/Rare/Very Rare/etc...)

Disabled

Multiple Computers

What if I add or remove item(s) from my SDB using a different browser or computer?

Simply view the item(s) in your normal SDB on the computer with the extension to update the information for the extension at any time.

Can I use the extension on multiple computers?

Yes, but you will need to add your items and folders on each separately. An option to sync folders with your Google account may be included in future versions.

Folders

Is there a limit on how many folders I can have?

No. You can have as many folders as you have items.

Can I put an item in multiple folders, or have some of an item in one folder and some of it in another?

No. Each item can only be in one folder, and moving one of an item to a folder moves all of that item.

PIN / Security

What if I have a PIN on my SDB?

If you have a PIN enabled on your SDB, the extension will not be able to remove items. This is intentional. If you'd like to remove items, you can either disable the PIN on your SDB while removing items, or use the standard SDB to remove them. All other features, such as adding items to the extension, sorting, and adding to folders should work normally.

Source Code

How can I build the extension from source?

  1. Get the source code and save it to your computer: SDB-Enhanced
  2. Visit chrome://extensions in your browser (or open up the Chrome menu by clicking the icon to the far right of the Omnibox and select Extensions under the Tools menu to get to the same place).
  3. Ensure that the Developer Mode checkbox in the top right-hand corner is checked.
  4. Click Load unpacked extension... to pop up a file-selection dialog.
  5. Navigate to the directory in which your extension files live, and select it.
Steps 2-5 are from Getting Started: Building a Chrome Extension on the Chrome Developers site.
All of the files in the lib/ dir can be replaced from their official sources: Bootstrap, jQuery, and jQuery UI.
